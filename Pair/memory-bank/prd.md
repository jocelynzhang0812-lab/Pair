# Feature Specification

## Feature: Onboarding

Goal: Create user profile.

Input: - Public profile URL - User information

Output: - Profile - Tags - Objectives

Done: User enters Today.

## Feature: Today

Goal: Show valuable matches.

Components: - Greeting - Agent status - Match cards

Match Card: - Avatar - Name - Role - Why match - Agent opening - Send -
Pass

Done: Send creates A2A session.

## Feature: A2A

Goal: Agents complete first communication.

Rules: - Two agents - Maximum 5 rounds each

Output: Summary

Done: Summary generated.

## Feature: Summary

Output: - Alignment score - Common topics - Risk - Next action

## Feature: Public Profile

Goal: Create shareable AI identity card.
