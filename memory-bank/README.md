# Pair Codex Memory Bank

## Goal

Build Pair MVP.

Pair is an AI Native Professional Relationship Network. AI helps users
discover valuable professional relationships.

## Development Rule

Before coding: 1. Read product.md 2. Read prd.md 3. Read ui-spec.md 4.
Read data-model.md 5. Read api-contract.md 6. Follow implementation.md

## Core Principle

AI assists relationship building. AI does not replace human
relationships.

## MVP Flow

Login → Profile Creation → Today Match → A2A Session → Summary → Public
Profile

## Out of Scope

-   Social feed
-   Search
-   Chat system
-   Meeting scheduling
-   Feedback system
